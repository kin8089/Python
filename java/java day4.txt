Day 4

	[생성자 메서드 overload(중복 정의)가능]

1. 생성자 메서드내에서 클래스내에 다른 생성자 메서드를 호출할 수 있습니다.

2. 생성자 메서드 내에서 다른 생성자 메서드를 호출 할 때는 생성자이름 대신 this()를 사용

3. 생성자 메서드 내에서 다른 생성자 메서드를 호출 할 때는 1번만 호출이 가능

4. 생성자 메서드 내에서 다른 생성자 메서드를 호출 할 때는 첫번째 문장에서만 가능



	[static 매서드]

static 매서드는 static 변수를 처리(process)하는 기능을 수행하는 목적

static 매서드는 객체 생성없이 클래스 이름으로 호출이 가능

static 매서드에서는 static 클래스변수 or static 매서드만 사용,호출 가능

   but non - static 매서드에서는 static변수 or static매서드 + 인스턴스변수, 인스턴스 매서드 모두 사용가능
